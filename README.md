# 2D-Floor-Plan-Images-to-3D-Model

r2v_clean.py
Conversion from raster to vector (Input: .png file, Output: .shp file)

refined_map.py
Conversion of floor plan vector to geojson file (Input: .shp file, Output: .geojson file)

doors_vector.py
Conversion of csv file with door locations to vector file (Input: .csv file, Output: .shp file)

convert_to_geojson_doors.py
Conversion of vector file for doors to geojson file (Input: .shp file, Output: .geojson file)

floor-plan-object-detection
Detection of doors in the floor plan image, takes image and gives csv with door locations (Input: .png file, Output: .csv file)
